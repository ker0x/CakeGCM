# CakeGCM

CakeGCM is a plugin for CakePHP to send downstream message to an Android or iOS device through Google Cloud Messaging

## Requirements

* CakePHP >= 3.*
* PHP >= 5.5

## Documentation

The documentation is available [here](http://cakegcm.readthedocs.org/en/latest/)
